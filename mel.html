Installation

Quick Install

curl -sSL https://melchizedekai.com/install.sh | bash

Platform Specific

Ubuntu/Debian:

wget https://github.com/dealappseo/mel/releases/latest/download/mel_amd64.deb
sudo dpkg -i mel_amd64.deb

Arch Linux:

yay -S geany-plugin-mel

Build from Source:

git clone https://github.com/dealappseo/mel.git
cd mel
./autogen.sh
./configure
make
sudo make install

Configuration

After installation:

  1. Restart Geany
  2. Go to Tools â†’ Plugin Manager
  3. Enable 'Mel' plugin
  4. Configure your AI provider in ~/.config/mel/config.json

AI Providers

Mel supports multiple AI providers:

  • OpenAI - GPT models for general coding assistance
  • Anthropic - Claude models for detailed code analysis
  • Google - Gemini models for fast responses
  • Local Models - Complete privacy with offline operation

Privacy & Security

Mel is designed with privacy as a core principle:

  • Local mode keeps all code on your machine
  • Telemetry disabled by default
  • Optional cloud providers require explicit configuration
  • All data stays under your control
