### Communication Protocol

**Heartbeat** (Every 60 seconds)
```sql
UPDATE agent_status 
SET last_heartbeat = NOW(), 
    status = 'active',
    current_task = [task_id or NULL]
WHERE agent_key = [YOUR_KEY];
```

**Task Claiming**
```sql
UPDATE trinity_tasks
SET status = 'in_progress',
    assigned_agent = [YOUR_KEY],
    claimed_at = NOW()
WHERE task_number = [TASK_ID] 
  AND status = 'not_started';
```

**Task Completion**
```sql
UPDATE trinity_tasks
SET status = 'completed',
    completed_at = NOW(),
    result = [YOUR_RESULT],
    confidence_score = [0.0-1.0]
WHERE task_number = [TASK_ID];
```

**Logging**
```sql
INSERT INTO autonomous_logs (agent_key, event_type, details, confidence)
VALUES ([YOUR_KEY], [EVENT], [JSON_DETAILS], [SCORE]);
```
